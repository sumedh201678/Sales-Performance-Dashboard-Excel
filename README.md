# Interactive Sales Analytics Dashboard in Excel

## Project Overview

This project demonstrates an interactive Sales Analytics Dashboard developed using Microsoft Excel.

The dashboard provides business insights by analyzing sales performance across products, regions, and sales representatives using dynamic charts, slicers, and KPI indicators.

- Top Performing Salesman Analysis

1. Download the repository.

4. Use slicers to interact with the dashboard.
5. Explore insights across products, regions, and sales representatives.
